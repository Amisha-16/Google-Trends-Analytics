Google Trends Data Science Assignment

Problem Statements

1. Write a code where by changing just the keyword, we can search for multiple things.

2. Find the Top 15 countries where the keywords are searched the most and create a visual representation.

3. Plot a world map showing the countries that search the keyword the most.

4. Extract the time-wise interest of the keyword and analyse how it trended over different years.

5. Compare related keywords and plot the graph.
