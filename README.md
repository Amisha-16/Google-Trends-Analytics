# Google-Trends-Analytics
Google Trends Data Analysis project answering real-world data-driven questions using PyTrends, Python, and data visualization techniques. Includes keyword comparison, country-wise trend analysis, world map visualization, and time-based interest insights.

# Google Trends Data Analysis 

This project focuses on analysing Google Trends data to understand how keywords perform across different regions and time periods.

# Objectives
This project answers the following questions:
1. Dynamic keyword search by changing only the search term.
2. Identification of the top 15 countries where the keyword is searched most, with visual representation.
3. World map visualization of countries based on keyword interest.
4. Time-wise trend analysis of the keyword across different years.
5. Comparison of related keywords with graphical representation.

# Technologies Used
- Python
- PyTrends
- Pandas
- Matplotlib
- Seaborn
- GeoPandas
